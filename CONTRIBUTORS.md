## 🤝 How to Contribute

Nous accueillons toutes les contributions ! Voici comment participer :

### 1. Signaler des bugs ou proposer des features
- Ouvrir une **issue** sur GitHub
- Fournir des détails (OS, version, logs)
- Proposer des solutions si possible

### 2. Contribuer au code
1. **Fork** le projet
2. Créer une branche: `git checkout -b feature/ma-feature`
3. Commiter: `git commit -m "Feat: Description"`
4. Push: `git push origin feature/ma-feature`
5. Ouvrir une **Pull Request**

### 3. Améliorer la documentation
- Corriger typos, clarifier sections
- Ajouter exemples, tutoriels
- Traduire en anglais (docs FR → EN)

### 4. Tester et donner du feedback
- Tester sur votre environnement
- Partager votre expérience (Discord, GitHub Discussions)
- Proposer améliorations UX
