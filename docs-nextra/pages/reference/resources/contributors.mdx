## Comment Contribuer ?

Vous souhaitez contribuer à CSWeb Community Platform ?

### 1. Code & Features

```bash
# Fork le repository
git clone https://github.com/BOUNADRAME/pg_csweb8_latest_2026.git

# Créer une branche
git checkout -b feature/ma-nouvelle-fonctionnalite

# Commit et push
git push origin feature/ma-nouvelle-fonctionnalite

# Créer une Pull Request
```

### 2. Documentation La documentation est en **Nextra (Next.js + MDX)** :

```bash
cd docs-nextra
npm run dev
# Éditer les fichiers .mdx dans pages/
```

### 3. Signaler des Bugs Ouvrez une issue sur GitHub :
- **Bug Report** : [https://github.com/BOUNADRAME/pg_csweb8_latest_2026/issues/new](https://github.com/BOUNADRAME/pg_csweb8_latest_2026/issues/new)
- **Feature Request** : Proposez vos idées
- **Documentation** : Signalez les sections à améliorer

### 4. Support & Questions

- **Discord** : [https://discord.gg/csweb-community](https://discord.gg/csweb-community)
- **Email** : contact@csweb-community.org
- **Twitter** : [@CSWebCommunity](https://twitter.com/CSWebCommunity)
