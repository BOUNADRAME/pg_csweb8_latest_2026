# Roadmap Feuille de route du projet CSWeb Community Platform.

## Vision Long Terme Démocratiser CSWeb pour tous les instituts statistiques africains avec une plateforme moderne, flexible et facile à déployer.

## v2.0 (Mars 2026) - COMPLÉTÉ

**Documentation & Architecture Flexible**

### Réalisé

- Documentation Nextra complète (47 pages)
- Architecture flexible (Local/Remote)
- Multi-SGBD (PostgreSQL/MySQL/SQL Server)
- Breakout sélectif (Assietou DIAGNE)
- Docker setup 5 minutes
- API Reference complète
- Validation production (RGPH5 Sénégal)

## v2.1 (Avril 2026) - EN COURS

**UI Moderne & Administration**

### Planifié

 - Jobs breakout automatiques

**Performance & Scalabilité**

- **Sécurité** :
