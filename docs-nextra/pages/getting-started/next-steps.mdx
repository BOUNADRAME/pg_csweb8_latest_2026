### RGPH5 Sénégal (Production INS)

Architecture remote avec séparation metadata/analytics :

- Base metadata : MySQL vanilla CSWeb (production ANSD)
- Base breakout : PostgreSQL remote (analytics ANSD)
- Breakout sélectif par région
- Scheduler pour breakouts nocturnes
