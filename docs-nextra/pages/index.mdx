## Qu'est-ce que CSPro et CSWeb ?

**CSPro** (Census and Survey Processing System) est le logiciel officiel du **U.S. Census Bureau** pour la collecte de données de recensements et d'enquêtes statistiques. Utilisé par plus de 100 instituts statistiques dans le monde.

**CSWeb** est le serveur web officiel qui accompagne CSPro pour recevoir, stocker et synchroniser les questionnaires collectés sur tablettes Android.

> **Note :** CSWeb vanilla (version officielle) est puissant mais complexe : installation manuelle MySQL, configuration Apache/PHP, aucune interface moderne.

## Innovations Principales

### Breakout Sélectif par Dictionnaire

Au lieu de "breaker" TOUS les dictionnaires (lent, risqué), vous pouvez maintenant cibler **un seul dictionnaire** à la fois.

```bash
php bin/console csweb:process-cases-by-dict EVAL_PRODUCTEURS
```

**RGPH5 Sénégal** : Le breakout sélectif a permis de traiter l'ensemble des questionnaires du recensement en une fraction du temps initialement prévu.

[Guide Complet](/guides/breakout/selective)

### Architecture Flexible Multi-DB

Choisissez votre base de données pour le breakout analytics :

- **PostgreSQL** : Recommandé pour analytics et requêtes complexes
- **MySQL** : Compatible avec CSWeb vanilla
- **SQL Server** : Pour écosystème Microsoft

**2 modes de déploiement** :
- **Local** : Tout sur une machine (dev/test)
- **Remote** : Serveurs distants (production RGPH)

[Architecture détaillée](/guides/architecture)
