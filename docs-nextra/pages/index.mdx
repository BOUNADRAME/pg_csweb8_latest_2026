## Qu'est-ce que CSPro et CSWeb ?

### CSPro (Census and Survey Processing System)

**CSPro** est le logiciel officiel du **U.S. Census Bureau** pour la collecte et le traitement de données de recensements et d'enquêtes statistiques.

<div className="grid md:grid-cols-3 gap-4 my-6">
  <div className="border border-gray-200 dark:border-gray-800 rounded-lg p-4 bg-blue-50 dark:bg-blue-900/20">
    <h4 className="font-bold mb-2">📱 Application Mobile</h4>
    <p className="text-sm text-gray-600 dark:text-gray-400">
      Collecte de données sur tablettes/smartphones Android
    </p>
  </div>
  <div className="border border-gray-200 dark:border-gray-800 rounded-lg p-4 bg-purple-50 dark:bg-purple-900/20">
    <h4 className="font-bold mb-2">📋 Questionnaires</h4>
    <p className="text-sm text-gray-600 dark:text-gray-400">
      Designer de formulaires avec logique et validations
    </p>
  </div>
  <div className="border border-gray-200 dark:border-gray-800 rounded-lg p-4 bg-green-50 dark:bg-green-900/20">
    <h4 className="font-bold mb-2">🔄 Synchronisation</h4>
    <p className="text-sm text-gray-600 dark:text-gray-400">
      Envoi des données collectées vers un serveur central
    </p>
  </div>
</div>

**Utilisé par** : RGPH (Recensements), enquêtes agricoles, EDS, MICS, et plus de 100 instituts statistiques dans le monde.

### CSWeb (CSPro Web)

**CSWeb** est le serveur web officiel qui accompagne CSPro pour :

<div className="space-y-2 text-base my-4">
  <div>✅ <strong>Recevoir</strong> les questionnaires envoyés depuis les tablettes CSPro</div>
  <div>✅ <strong>Stocker</strong> les données dans une base MySQL</div>
  <div>✅ <strong>Gérer</strong> les utilisateurs et les dictionnaires CSPro</div>
  <div>✅ <strong>Synchroniser</strong> les mises à jour bidirectionnelles</div>
</div>

<Callout type="info">
**CSWeb vanilla** (version officielle de csprousers.org) est **puissant mais complexe** à configurer : installation manuelle MySQL, configuration Apache/PHP, aucune interface moderne.
</Callout>

## 🌟 Innovations Principales

### 1. Breakout Sélectif par Dictionnaire

<div className="border-l-4 border-purple-500 bg-purple-50 dark:bg-purple-900/20 p-6 my-6 rounded-r-lg">
  <h4 className="text-lg font-bold mb-3 text-purple-900 dark:text-purple-100">
    🌟 Innovation : Assietou DIAGNE (ANSD, Sénégal)
  </h4>

  <p className="mb-4 text-purple-900 dark:text-purple-100">
    Au lieu de "breaker" TOUS les dictionnaires (lent, risqué), vous pouvez maintenant cibler <strong>un seul dictionnaire</strong> à la fois.
  </p>

  <div className="bg-white dark:bg-gray-900 p-3 rounded-lg border border-purple-300 dark:border-purple-700 mb-4 font-mono text-sm">
    php bin/console csweb:process-cases-by-dict EVAL_PRODUCTEURS
  </div>

  <p className="text-sm text-purple-800 dark:text-purple-200">
    <strong>Impact RGPH5 Sénégal</strong> : 3.2M questionnaires breakés en <strong>6h au lieu de 48h</strong> 🚀
  </p>
